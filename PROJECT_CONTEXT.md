# SoundAtlas Project Context

This is a music social app inspired by Letterboxd, Spotify, and Apple Music.

Stack
- Next.js
- React
- Tailwind
- MusicBrainz API
- Cover Art Archive

Features built
- Artist search
- Artist pages
- Album pages
- Track lists
- 1–5 star ratings
- Review text boxes
- User profiles
- Profile review feed
- Homepage
- Album covers

Goal
A social platform for reviewing and rating music like Letterboxd.

Next features
- Activity feed
- Following users
- Trending albums
- Global ratings
